Laya Pass 隨行券 消費者權益說明


 

1. 只有 Laya NOW 會員才能購買 Laya Pass 隨行券嗎？

是的。為了確保您的權益與票券安全，Laya Pass 隨行券的所有服務（包含購買、兌換、查詢餘額、轉贈親友），皆需登入 Laya NOW App 會員後方可使用。

2. 購買 Laya Pass 隨行券時，可以累積會員點數？

可以。

  • 購買時： 系統將依據消費金額依據Laya NOW會員點數發放原則，立即發放會員點數。

  • 兌換時： 因購買時已發放過點數，故兌換商品時不再重複集點。 

3. 兌換或送禮後， Laya NOW App內的剩餘數量會同步更新嗎？

會的。系統將即時更新數據，您每次兌換或轉贈完成後，剩餘張數都會同步扣除並更新。

4. Laya Pass 隨行券有使用期限嗎？

有的。Laya Pass 專區內的商品皆有標示兌換期限。請您務必留意商品頁面說明，並於期限內兌換完畢。

5. 如何查詢購買、兌換或送禮記錄？

您可隨時於 App 內查詢完整紀錄：

  • 購買紀錄： 點選功能主選單中的「會員」→「歷史訂單」→「隨行券紀錄」，即可查看購買明細。

  • 兌換/送禮紀錄： 點選功能主選單中的「會員」 → 點擊任一張「Laya Pass 隨行券」，即可查閱該票券的詳細兌換、送禮或受贈歷程。

6. 全台拉亞漢堡門市都可以兌換 Laya Pass 隨行券嗎？

適用門市以 Laya NOW App 內顯示為準。 目前大部分門市皆可兌換，惟以下特定門市暫不適用：
金門金城、澎湖三多、龜山華亞科、蘆竹鴻佰、新店HTC總部、比佛利、迴龍光啟、北市松德、三峽中山、三峽中華、霧峰亞洲大學、羅東中山、北市懷德、埔里民生、北市中華、東勢中山、台北八德、八德介壽、南崁台茂、楊梅環東

7. 我可以將 Laya Pass 隨行券送給朋友嗎？ 

可以。購買後您可以自由轉贈給好友，但受贈者（您的好友）也必須是 Laya NOW 會員，才能成功接收與使用票券。

8. 如果想退貨，退貨金額如何計算？

自購買商品進入會員帳戶之次日起算7 日內皆可受理退貨申請，請來信客服信箱 layaservice@sunspark-group.com 辦理。

  • 全額退費：若您完全未使用（未兌換任何商品），經確認無誤後將全額退費。

  • 部分退費： 若您已兌換過部分商品，退費計算將以「原支付金額」扣除「已兌換商品之門市原價（非優惠價）」後，退還剩餘款項。若扣除後餘額為零或負數，則無法退費。

9. 退貨申請成功，多久會收到退款？

公司確認退貨申請後，將於15個工作日內完成內部退款及辦理交易取消手續，實際退款時間須由您自行與發卡銀行確認。 

10. 票券過期了但沒用完，可以退款嗎？

系統將自動退款。若您的票券在「兌換期限」屆滿後仍有剩餘張數，系統將主動進行退款作業。

  • 退款計算： 依據您當初支付的金額與點數，扣除已使用價值後計算。

  • 退款方式： 款項將自動退回您購買時所使用的原支付工具（如信用卡刷退）。

四、退貨及退款說明

1、商品未逾兌換期限退貨：

消費者得於本服務購買商品進入會員帳戶之次日起算7天內（含例假日）自行洽客服申請依購買支付金額計算之退貨退款作業，退款時已兌換商品將以原價扣抵(商品原價依隨行券面載明為主)，並於扣除已兌換商品金額後，退回原支付帳戶。另本服務依隨行券結帳金額計算發放之 Laya NOW 會員點數(滿50元贈1點)，同步依最終退款金額計算溢領點數回收，退款計算及點數回收方式詳如下述。

2、商品逾兌換期限退貨：

逾兌換期限之商品，系統將自動依據退款計算及點數回收方式計算剩餘價值，逕行將款項退回原支付帳戶，並同步回收對應退款金額之溢領點數，會員無須另行申請。如同一筆訂單內包含不同兌換期限之商品，系統將待該筆訂單內「最長使用期限之商品」屆滿後，始自動執行整筆訂單之剩餘金額核算與退款；針對訂單內較早逾期之商品，將暫不執行個別退款，須待整筆訂單效期全數屆滿後一併處理，相關計算方式詳如下述。

3、計算方式：

退款之計算(如退款金額小於0者，以零元計，不再做退費和收費)：

實付金額(原購買支付金額)－【已兌換商品原價×已兌換商品個數】＝退款金額。

點數之回收：

原購買隨行券獲得點數－【(已兌換商品原價×已兌換商品個數)÷50】(採小數點後無條件捨去計算)＝回收點數

試舉例：

大杯奶茶每杯原價30元，一組買10送1(共計11杯)優惠價為300元，購買完成即贈送6點(滿50元贈1點)。

購買組數為1組，以最小單位之「商品個數11件」計算退貨退款，試算結果如下，並以此類推：

未兌換、欲全部退貨：     300-(30×0)=300，故退款300元；6-(30×0÷50)=6-0=6，故回收6點。

已兌換1件、欲退貨10件：300-(30×1)=270，故退款270元；6-(30×1÷50)=6-0=6，故回收6點。

已兌換2件、欲退貨9件：300-(30×2)=240，故退款240元；6-(30×2÷50)=6-1=5，故回收5點。

已兌換3件、欲退貨8件：300-(30×3)=210，故退款210元；6-(30×3÷50)=6-1=5，故回收5點。

已兌換4件、欲退貨7件：300-(30×4)=180，故退款180元；6-(30×4÷50)=6-2=4，故回收4點。

已兌換5件、欲退貨6件：300-(30×5)=150，故退款150元；6-(30×5÷50)=6-3=3，故回收3點。

已兌換6件、欲退貨5件：300-(30×6)=120，故退款120元；6-(30×6÷50)=6-3=3，故回收3點。

已兌換7件、欲退貨4件：300-(30×7)=90，故退款90元；6-(30×7÷50)=6-4=2，故回收2點。

已兌換8件、欲退貨3件：300-(30×8)=60，故退款60元；6-(30×8÷50)=6-4=2，故回收2點。

已兌換9件、欲退貨2件：300-(30×9)=30，故退款30元；6-(30×9÷50)=6-5=1，故回收1點。

已兌換10件、欲退貨1件：300-(30×10)=0，故退款0元；6-(30×10÷50)=6-6=0，故回收0點。

無論購買幾組，應退款金額，皆以原購買支付金額(優惠價)扣除已兌換之商品原價，如同上述試算範例，請確認同意後再進行購買，付款成功後表示已詳閱且同意本退貨退款暨點數回收計算方式。

4、會員辦理退貨須回收之 Laya NOW 會員點數，如帳戶內點數餘額不足扣抵，系統將允許點數呈現負值；惟該會員帳戶在補足點數至正值(大於或等於0 )前，本公司將暫停購買 Laya Pass 隨行券之權限，直至補足完成後始得恢復使用。

5、本公司將於確認退貨申請後，於 15 個工作日內完成內部退款及辦理交易取消手續。惟實際退費款項入帳日期，須視發卡銀行或電子支付平台作業時程為準；如採信用卡退刷，款項可能遞延至次月帳單，請同步留意您的帳戶或帳單資訊。
