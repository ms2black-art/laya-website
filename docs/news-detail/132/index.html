1. 集龍珠任務活動僅限登入LAYA NOW APP會員參與，於未登入會員情況下恕不適用。

集龍珠任務活動中每間門市皆有自己的集龍珠任務卡，系統會自動判斷消費當下所在門市，並將龍珠印花發放至指定門市任務卡中，門市間的龍珠印花不得轉移亦不得跨店兌換贈品。
3. 單筆消費使用【早點到】服務，除消費滿100元可獲得1枚龍珠印花外，再加碼贈送1枚龍珠印花，不累計贈送，每筆訂單贈送上限為1枚龍珠印花，消費未滿100元恕不適用。
4. 單筆消費使用【預約點餐】功能，除消費滿100元可獲得1枚龍珠印花外，再加碼贈送1枚龍珠印花，不累計贈送，每筆訂單贈送上限為1枚龍珠印花，消費未滿100元恕不適用。
